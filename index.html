A simple browser extension for Etsy sellers that displays useful listing insights directly inside the Etsy interface.

What This Extension Does

EtsyHelper is a seller tool designed to help Etsy shop owners quickly view important listing information without opening each listing manually.

After a secure OAuth connection with Etsy, the extension reads approved shop and listing data and displays a small information card directly below each product or listing inside Etsy.

Each card is meant to help sellers review listing performance faster while staying inside the Etsy interface.

Listing Creation Date

Quickly see when a listing was originally created.

Last Update Date

View when the listing was last updated or changed.

Favorites Count

See how many users have favorited a listing.

Sales Count Per Listing

Display sales insights for each listing when available through approved access.

The purpose of EtsyHelper is to save sellers time and make listing analysis easier with clear, useful information shown directly on Etsy pages.

Secure Connection and Trust

EtsyHelper uses official Etsy OAuth for account connection. Users sign in through Etsy, not through this extension.

EtsyHelper is designed to provide read-only listing insights and does not create, edit, delete, or modify listings inside a seller's shop.

The extension is built to request only the minimum access needed to display listing insight features for the authenticated seller.

How It Works

The user installs the browser extension and securely connects their Etsy account using Etsy OAuth. After authorization, the application retrieves approved data through Etsy's API and adds a small information card below each product or listing inside the Etsy interface.

This card helps sellers quickly review listing details such as creation date, last update date, favorites, and sales per listing without leaving the page.

  1. Install the extension
  2. Connect securely with Etsy
  3. Approve access through Etsy's official login flow
  4. Open Etsy listing pages
  5. View an insight card below each product or listing
Typical use case

A seller opens Etsy listing pages, and EtsyHelper adds a simple card under each listing to make review and comparison faster.

What We Access

EtsyHelper only requests the permissions needed to provide its core features.

  • Listing metadata needed to show creation and update dates
  • Listing favorites data when available through approved access
  • Authorized seller account access through Etsy OAuth
  • Approved data required to calculate or display listing insights

What We Do Not Collect

  • We do not collect your Etsy password
  • We do not ask you to paste your own Etsy API key
  • We do not ask for your shared secret
  • We do not create, edit, or delete your listings
  • We do not change your shop settings

Privacy Policy

EtsyHelper only requests the permissions needed to provide its core features. The application is intended to access approved Etsy data required to display listing insights for the authenticated seller.

The application does not ask users to create or share their own Etsy API key. Access is handled through Etsy OAuth authorization.

User data is used only to provide the application's features and improve the seller experience. Sensitive account access should be stored and handled securely.

EtsyHelper aims to collect only what is necessary for core functionality. Users should be able to disconnect their Etsy account and revoke access at any time.
